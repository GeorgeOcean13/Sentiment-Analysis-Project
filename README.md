Sentiment Analysis Project:
This project analyzes text reviews and classifies them as Positive, Negative, or Neutral using Machine Learning.

It uses:
- TF-IDF Vectorizer for feature extraction
- Logistic Regression for prediction
- Streamlit for building the user interface

Files:
- train.py → Trains and saves the model
- app.py → Runs the Streamlit web app for predictions
- data/ → Contains the dataset (Reviews_small.csv)

How to run:
1. Train the model: python train.py
2. Start the app: streamlit run app.py
